Guía de Parámetros
Tasa de Aprendizaje
  • 0.1 - Valor recomendado para la mayoría de casos
  • 0.01-0.05 - Para datos complejos o inestables
  • 0.5-1.0 - Para problemas simples como compuertas lógicas
Número de Iteraciones
  • 100-500 - Para problemas simples
  • 1000-5000 - Para datos más complejos
  • El entrenamiento se detiene automáticamente al converger
Error Máximo Permitido
  • 0.1 - Valor máximo permitido (10% de error)
  • 0.05 - Para mayor precisión (5% de error)
  • 0.01 - Para máxima precisión (1% de error)
  • 0.001 - Para precisión extrema (0.1% de error)
  • El entrenamiento se detiene cuando el error es menor o igual a este valor (máximo 0.1)
Selección de Columnas
  • Entrada (X): Características que usa el perceptrón para predecir
  • Salida (Y): Variable objetivo que queremos predecir
  • El perceptrón simple solo maneja una variable de salida
  • Puede manejar múltiples variables de entrada
